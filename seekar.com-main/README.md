# 🚀 Seekar - Engineering Job Portal

**Seekar** is a vibrant, two-sided marketplace designed specifically for the engineering community. It connects aspiring engineers (Candidates) with leading companies (Employers) to streamline the hiring process for internships and full-time roles.

## 🌟 Key Features
* **For Candidates:** Explore jobs across IT, Mechanical, Civil, and Electronics branches with one-click applications.
* **For Employers:** A dedicated dimension to post job vacancies and manage talent search.
* **Smart Filtering:** Advanced search and filter options for branch-specific roles.
* **Modern UI:** A colourful, responsive interface built for high-speed navigation.
